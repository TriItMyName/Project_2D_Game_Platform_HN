# Project_2D_Game_Platform_HN
Overview

This is a 2D platformer game project developed using Unity and C#. The main goal of the project is to build a complete game while applying knowledge of game programming, gameplay design, and interaction mechanics in a 2D environment.
In the game, players control a character to overcome levels, avoid obstacles, fight enemies, and progress through increasingly challenging stages.

Key Features
- 2D platformer character movement (running, jumping, and environment interaction)
- Basic combat system with enemies
- Level design with various challenges and terrains
- Physics-based mechanics using Unity for movement and collision handling
- In-game save system for player progression
